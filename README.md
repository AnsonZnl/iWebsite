# iWebsite

通过Next.js构建的网站，可用于企业官网、毕业设计。

## 使用技术

- Create-Next-App
- Next.js
- Tailwind CSS
- Vercel
- next-connect
- React-Qeury
- Vercel KV
- Vercel Postgres
- Prisma
- JWT

## 快速开始

请你预先安装好NodeJS(v16+)，clone此项目到您的本地，然后运行一下命令启动服务:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000) 即可看到页面


## 优点
- 利于 SEO 搜索引擎收录
- 加快首屏呈现时间
- 同时拥有​​单页(SPA)​​​和​​多页路由​​的用户体验
